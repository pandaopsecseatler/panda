# Rivals Assistant – Premium Roblox Exploit HUD & Alert System

Advanced client-side assistant for **Rivals** on Roblox with target player detection, beautiful animated UI, Discord webhook alerts, premium key system, and auto-updating leaderboard.

## ✨ Core Features

- **Target & Priority Player Detection**  
  Instant popup notifications + Discord webhook when specific players (or high-priority targets) join your server

- **Beautiful Animated Popups**  
  Smooth slide-in banners with player avatar, glowing outline, shine effect, and auto-dismiss

- **Real-time Leaderboard HUD**  
  - Draggable & resizable window  
  - Pulsing rank icons with glow  
  - Shows Level, Win Streak, ELO + detailed rank (Bronze 1–3, Silver 1–3, ..., Archnemesis)  
  - Auto-refreshes every few seconds  
  - Top 3 places highlighted in gold/silver/bronze colors  
  - Toggle with **F5** | Full shutdown with **End**

- **Premium Key System** (HWID-locked)  
  - Beautiful animated key entry screen with blur + particles  
  - Verifies keys against your own Discord bot / Render-hosted API  
  - Auto-saves valid key locally  
  - Periodic background re-verification

- **Modern Theme System**  
  - Dark (default), Light, Neon, Colorblind presets  
  - Live theme switching from settings panel  
  - Auto-saves your preference

- **Discord Webhook Integration**  
  - Sends rich embeds with:  
    - Player rank emoji & ELO  
    - Level / Win Streak  
    - Priority/Target status  
    - Ping role support  
  - Cooldown protection to avoid spam

- **Panda Intro Splash Screen**  
  - Cool animated cat logo spin + zoom + glow  
  - Subtle watermark ("panda made this")  
  - Toggleable in config

- **Quality of Life Extras**  
  - Auto-save settings (theme, toggles, etc.) to JSON  
  - Notification popups when settings are saved  
  - Mute alerts option  
  - Whitelist support (never show certain players in leaderboard)  
  - Clean border removal across all UI elements

## Controls / Hotkeys

| Key       | Action                            |
|-----------|-----------------------------------|
| **F5**    | Toggle leaderboard visibility     |
| **End**   | Emergency shutdown (destroy all)  |
| **Gear icon** | Open/close settings panel     |
